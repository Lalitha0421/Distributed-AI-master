# 🚀 Distributed Multi-Agent AI Knowledge Assistant

A professional, multi-agent RAG (Retrieval-Augmented Generation) system built with:
- **Frontend**: React + Vite + Framer Motion (Glassmorphism UI)
- **Backend**: FastAPI + LangGraph + Groq (Llama 3.1)
- **Vector DB**: ChromaDB with Hybrid Search (BM25 + Vector)

## ✨ Features
- **Intelligent RAG**: Combines exact keywords and meanings for high accuracy.
- **Agentic Tracing**: Real-time view of what each AI agent is "thinking."
- **Dashboard**: Real-time performance metrics and self-improvement logs.
- **Smart Sync**: Automatically cleans up knowledge if files are deleted.

## 🛠️ Tech Stack
- **AI**: Groq, Sentence Transformers (MiniLM)
- **Infrastructure**: Docker, Nginx, Python 3.11
- **Optimization**: Hybrid Search & Cross-Encoder Reranking
